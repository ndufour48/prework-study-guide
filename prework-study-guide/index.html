HTML

  • HTML, or Hypertext Markup Language, gives the content structure
  • !DOCTYPE html is the declaration that's placed at the top of the file that let's the broswer know to expect HTML on the page
  • html lang="en" is an opening tag that tells the browser the language for the webpage. The atribute "en" is for english in this case
  • The head element contains metadata that helps the browser render the page. Elements within the head are not visible when a user looks at the webpage
  • The title element defines the title for a webpage, and is extremely important for Search Engine Optimization, or SEO
  • The style element defines the CSS style associated with a webpage
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The meta element defines the metadata such as the character set, describtion, keywords, author, and viewport. -- more about other types of information that can be stored in the meta tag here
  • The script element embeds client-side scripts in the HTML document
  • The body element contains the content that is shown to the user, to see and interact with
  • The h1, h2, h3, etc. are heading tags
  • The p tag is used to wrap most of the text on a webpage
  • The ul tag is for unordered lists
  • The ol tag is for ordered lists
  • The li tag is for list items
  • The img element contains information about the images that are displayed on the webpage
  • The br tag creates a line of empty space, or a line break
  • The a tag, or anchor, creates a link to a webpage

CSS

  • CSS, or Cascading Style Sheets, gives the content style
  • A selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations contain the CSS property we want to apply and the value of the property
  • CSS box model - each box consists of margins outside the element, and the element's borders, padding, and content. -more on the box model here
  • The * selector acts as a wildcard symbol, and declares all the elements on the page will have the style
  • Make sure class attribute has been assigned to the correct selection elements in the markup
  • more CSS properties and values here

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds modifications in the current working branch to the staging area, -A indicates that we want to add all changes
  • git commit -m "commit message": commits the changes and adds a message that describes what the commit contains
  • git pull origin main: checks to see if local branch is in sync with the base branch
  • git push origin feature/starter-code: pushes our changes to the remote branch

JavaScript

  • JS, or JavaScript, is the logic and allows users to interact with the website
